# Drone Analysis – Battery System Model

This project simulates a photovoltaic (PV) system combined with different battery configurations and evaluates the long-term economic performance over 25 years.

The PV production model is identical to the simulation used in `PV_simulation_code.py`.

The model compares:
- BAU
- PV with a big battery charging the drone batteries
- PV with a total of 40 drone batteries batteries
